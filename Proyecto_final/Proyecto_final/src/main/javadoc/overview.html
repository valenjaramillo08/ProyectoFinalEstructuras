Plataforma academica de billeteras digitales construida con Spring Boot y MongoDB. El sistema modela usuarios, billeteras, transacciones, recompensas, alertas, operaciones programadas y analitica basada en estructuras de datos.

La documentacion describe las capas principales del backend: API REST, servicios de aplicacion, modelos de dominio, estructuras en memoria, persistencia MongoDB e infraestructura programada.
